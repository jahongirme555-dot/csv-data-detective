# CSV Data Detective Tool (DD Clues)

## Overview
This Python-based tool analyzes CSV datasets to assist with completing DD clues.

## Features
- CSV data preview
- Descriptive statistics (numeric & categorical)
- Anomaly detection by column
- Unique counts for categorical data
- Filtering and conditional queries
- Group by and aggregation (sum, mean, count)
- Date/time trend analysis
- Data visualizations (histograms, line charts)
- Export of transformed data (anomalies)

## Technologies
- Python
- pandas
- numpy
- matplotlib

## How to Run
```bash
python dd_tool.py your_file.csv
